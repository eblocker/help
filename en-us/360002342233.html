Release notes 0.8.3

 

Workaround for FritzBox 7490 issues

eBlocker now uses IP masquerading in manual network configuration. This ensures that SSL, FTP and other non-HTTP connections are more reliable when eBlocker is connected to a Fritzbox 7490.

Network configuration is verified

Input on the network configuration screen is now validated more thorougly.

Registration can be reset

The registration can be reset when eBlocker was registered with a wrong license key. Use this feature only if advised by support. This action can not be undone and you can not re-register the previous license key.

Note that the following reboot will take up to 5 minutes.

Improved tracker blocking

An issue was fixed where links from Google were sometimes leading to white pages.

UI improvements

The controlbar rendering was improved for Microsoft Internet Explorer and Chrome browsers.
