Check the network connection with Windows.

You can easily test the eBlockers connection with Windows.

If you do not know your eBlocker's IP address, please check the network settings of your router. There you should be able see the eBlocker and its IP address.

Please open the Windows Command Prompt called cmd.exe and type in the following command: tracert www.eblocker.com

This command shows you the way from your PC through the Internet to our website. Your eBlockers IP address should be in the first place followed by your routers IP address.
