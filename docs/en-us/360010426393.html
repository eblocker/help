No eBlocker Icon and no Controlbar

The eBlocker Pro and the eBlocker Family can only display the eBlocker Icon and the eBlocker Controlbar on websites when the eBlocker HTTPS function has been activated and the eBlocker certificate has been stored.

The eBlocker Base cannot display the eBlocker Icon or the eBlocker Controlbar on websites.

Further information about the eBlocker HTTPS function can be found here: HTTPS Function (SSL/HTTPS)

The eBlocker Dashboard is your control center and replaces all your usual functions of the eBlocker Controlbar.

Further information about the Dashboard can be found here: The eBlocker Dashboard
