Help, a website will no longer be displayed.

The more blocker lists you use, the more likely it is that a website will not be loaded by the browser.

For example, suppose you call the website facebook.com in your browser and then see the message "page cannot be loaded", then pause the eBlocker in the eBlocker Dashboard and try to load this page again. If this works, it may be due to one of the imported blocker lists.

Now end the pause of the eBlocker in the eBlocker Dashboard again. On the eBlocker Dashboard you will find the card "Blocker". Here you can now allow the website (domain).

The next time you access the website, you should be able to see it again.
