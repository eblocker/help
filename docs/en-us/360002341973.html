eBlocker Help > FAQs > Anon function (Tor-network)

The real IP address is displayed.

If an IP address will not be anonymized by the eBlocker, this may have several reasons.

  1. The eBlocker is not activated for the device.
  2. The eBlocker is enabled for the device, but the eBlocker HTTPS function was not enabled for the device and (or) the eBlocker certificate was not properly stored on the device.
  3. You were using neither the Tor network nor a VPN server connection. The IP address cannot be anonymized.
  4. Parts of the website may come from the browser cache.
  5. You are using a browser extension which could reveal the true IP address (for example FLASH).
  6. You may have activated the IPv6 protocol for your internal network (home network), which is not yet compatible with the eBlocker.
