What is fragFINN?

Starting with eBlockerOS version 2, we have added some extra protection for your children with an extended protection list from fragFINN. fragFINN is an German Internet search engine for children whose results have been specially selected for the young Internet users.

For e.g., when you create a new Parental Control profile, you can now restrict access restrictions as followed:


  • Select the main policy "Following categories are allowed, everything else is restricted".
  • Select fragFINN and save the setting.



Additionally we have added a fragFINN search field to the eBlocker message.
If you have set your eBlocker interface to English, you will see the search engine DuckDuckGo here.
