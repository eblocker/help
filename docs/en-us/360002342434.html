Release Notes 0.9.1

This maintenance release mainly includes bug fixes and improvements in the background.

Improvements and Bug Fixes

  • User interface improvements
  • Improvements for recognizing tracking links
  • Several bug fixes for SSL/HTTPS support
  • Bug fixes in the network configuration: duplicate IP-address assignment, setting the gateway
  • Internet Explorer is fully supported now

Known Issues

  • Microsoft Edge is not showing the eBlocker icon even though the eBlocker is working correctly. Workaround: Please use a different browser to access the eBlocker settings.
