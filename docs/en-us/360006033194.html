I can't see the eBlocker icon on all websites

There are two reasons why you may not see the eBlocker icon on all websites.

  1. eBlocker Base can not display the eBlocker icon on webpages. All features of your eBlocker Base are still active; only the icon is missing.
  2. If you own an eBlocker Pro or Family, you can use the SSL feature. To do this, please store the eBlocker certificate and activate SSL. The eBlocker icon will be visible on HTTPS pages afterwards.

If the icon is not displayed, you can access your eBlocker settings at any time by going to the eblocker.box page in your browser.
