How to integrate my own DNS server?

With the eBlockers DNS function you not only have the possibility to integrate external DNS servers, you can also integrate your own Do It Yourself (DIY) DNS server.

If you already run a DIY DNS server, you should first make sure that it is stored in your router as the DNS server. Now activate the DNS function in your eBlocker under Settings > DNS Firewall. Select the "Internet Provider" option.
