Clearing the browser cache (Windows and macOS).

We recommend to clear the browser cache after every eBlockerOS update - as well as to clear it occasionally. This can be done with all browsers either via the browser settings or with a simple key shortcut.

Here is a list of browsers and keyboard shortcuts for Windows and macOS.

Opera:

Windows: [Strg] + [Shift] + [Del]

macOS: [Cmd] + [Shift] + [Del]

You should not clear passwords, autofill form data and maybe third party service data.

Vivaldi:

Windows: [Strg] + [Shift] + [Del]

macOS: [Cmd] + [Shift] + [Del]

You should not clear passwords, form autofill data and maybe applications cache.

Chrome and Chromium:

Windows: [Strg] + [Shift] + [Del]

macOS: [Cmd] + [Shift] + [Del]

You should not clear passwords, autofill form data , hostet app data and media licenses.

Firefox and Cliqz:

Windows: [Strg] + [Shift] + [Del]

macOS: [Cmd] + [Shift] + [Del]

You should not clear active logins, offline website data and site preferences.

Edge:

[Strg] + [Shift] + [Del]

You should not clear form data, passwords, media licenses, pop-up exceptions, full screen permissions and notification permissions.

Internet Explorer:

[Strg] + [Shift] + [Del]

You should not clear form data, passwords and tracking protection.

macOS Safari:

macOS: [Cmd] + [Alt] + [E]
