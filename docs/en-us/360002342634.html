DHCP Server - There can only be one!

If you activate the DHCP server of your eBlocker, please remember to deactivate the DHCP server of your router aswell.

There can only be one DHCP server in the network.

If you deactivate the DHCP server of your eBlocker, please remember to reactivate the DHCP server of your router.

When changing the DHCP server, all devices in the network must be disconnected and reconnected once. Either disconnect the network cable for a short time, deactivate the WiFi for a short time, or restart the device.
