eBlocker Help > FAQs > HTTPS function (SSL/HTTPS)

Generating the certificate takes a long time.

You have activated the HTTPS function and see an "Attention" message. At the bottom right you will find the button "Enable HTTPS". This is grey, which was a known UI bug in older versions of eBlockerOS. You can click on the button and do not have to wait for the certificate to be created.
