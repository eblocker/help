Public IP has changed

If the public IP address of your router changes and you do not use a DynDNS service, the eBlocker Mobile function must be reset in the "eBlocker Mobile" menu. After that you can change the settings of the eBlocker Mobile function and have to store the eBlocker Mobile configuration for each device again in the OpenVPN App.
